[b]Command mode[/b]
Command mode technically doesn't have keybinds, but it will react to two specific keypresses.

[table=2]
[cell][code]<ESC>[/code][/cell][cell]Enter insert mode and delete command[/cell]
[cell][code]<Enter>[/code][/cell][cell]Execute command[/cell]

[cell][code]<DownArr>[/code][/cell][cell]See next command in command history[/cell]
[/table]
